Responsive Image Gallery

This page demonstrates the use of media queries to create a responsive image gallery. The layout adjusts based on the screen size.

  • On smaller screens, images are displayed in a single column.
  • At 772px width, the images are arranged in a two-column grid, with the last image spanning both columns.
  • At 992px width, the images are displayed in a two-column grid, with every third image spanning both columns.

The images are set to be responsive, ensuring they fit within their containers while maintaining aspect ratio.
